# NovelForge

你有過這種經驗嗎？腦中浮現一個絕佳的故事點子，打開空白文件，然後……盯了三小時，只寫出兩行。

NovelForge 就是為了解決這件事而生的。你只需要給它一個主題概念，它會幫你把世界觀搭好、角色捏出來、大綱理清楚、場景拆分明，最後逐章把小說寫出來。整個過程你隨時可以介入編輯——AI 是你的寫作夥伴，不是代筆機器。你在駕駛座，AI 幫你踩油門。

## 它能做什麼

- **多部小說同時進行** — 靈感到處噴？開多個專案各自管理，互不干擾
- **世界觀一鍵生成** — 丟一個主題給它，它還你類型、地理、世界規則，有模有樣
- **角色自動長出來** — 從名字、性格到背景故事，AI 會根據你的世界觀生出有血有肉的角色
- **大綱不用自己想破頭** — 故事摘要、章節標題一口氣出來，順序不滿意還能拖曳調整
- **章節拆成場景** — 每章自動拆成 3-5 個場景，不再對著空白頁發呆
- **逐章寫出完整內容** — AI 幫你產出 2000-4000 字的章節，還自動附上摘要讓下一章有上下文
- **沉浸式閱讀模式** — 寫完直接切到閱讀模式，享受自己的小說
- **每個模組可獨立調校** — 世界觀用 GPT-4o、角色用 Claude、章節用自架模型？隨你配，沒設定的自動吃全域預設
- **Mock 模式免 API Key** — 想先玩玩看？開 Mock 模式，內建範例故事直接跑一輪

## 快速開始

需要 Docker。不需要 API Key（可以先開 Mock 模式）。

```bash
git clone <repo-url> && cd novel

# 想玩真的 → 設 API Key
export OPENAI_API_KEY="sk-your-key"

# 想先試水溫 → 開 Mock 模式
export MOCK_AI=true

docker compose up -d
```

打開 http://localhost:5173，開始寫你的第一本小說。
